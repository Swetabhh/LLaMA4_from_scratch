# LLaMA4 From Scratch

A from-scratch implementation of Meta's Llama 4 architecture in pure Python.
Built to deeply understand how one of the most powerful open-source LLMs
works under the hood — not just how to use it.

## What's Implemented

- **Tokenizer** — text encoding and decoding pipeline
- **Attention Mechanism** — multi-head attention as used in Llama 4
- **Feedforward Layer** — the MLP block inside each transformer layer

## Why

Most people use LLMs as black boxes. This project is my attempt to
understand every component from first principles — by building it myself.

## Stack
- Python
- PyTorch
